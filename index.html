IS 312: Web Design and Programming

School of Technology & Computing

Credits: 5 Credits
Level: Undergraduate
Total Points: 1,000
Duration: 10 Modules

Course Description

This course introduces students to the strategic importance of web design and teaches students how to apply standards, design principles and web development tools to create effective business-driven websites. The course emphasizes a business-driven design, build, and test cycle, and introduces students to the fusion of multiple data sources into a website.

Technologies Covered

HTML CSS JavaScript TypeScript

Learning Outcomes

  • Understand HTML, CSS, JavaScript, and TypeScript
  • Apply HTML, CSS, JavaScript, and TypeScript to web applications
  • Analyze TypeScript for web applications
  • Evaluate TypeScript technologies for web applications
  • Create a web application by using TypeScript
